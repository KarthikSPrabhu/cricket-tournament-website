# Cricket Tournament Website

A comprehensive cricket tournament management system built with MERN stack.

## Features

- Player Registration with T-Shirt details
- Admin Management System
- Live Auction with Real-time Bidding
- Team Management
- Live Match Scoring
- Tournament Statistics
- Real-time Updates

## Tech Stack

- **Frontend**: React.js, Vite, CSS3
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Real-time**: Socket.io
- **File Upload**: Cloudinary/Multer

## Setup Instructions

### Prerequisites
- Node.js (v14 or higher)
- MongoDB
- Git

### Installation

1. Clone the repository
2. Setup backend
3. Setup frontend
4. Configure environment variables
5. Run the application

For detailed setup instructions, check [SETUP.md](docs/SETUP.md)
